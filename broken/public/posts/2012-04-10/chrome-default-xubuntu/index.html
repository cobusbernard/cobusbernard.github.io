Chrome as default browser in Xubuntu

Apr 10, 2012 in LINUX
linux config xubuntu
1 min read

If you need to set Chrome as the default browser in Xubuntu, run the following from your terminal:

gconftool-2 --type string -s /desktop/gnome/url-handlers/http/command "google-chrome %s"
